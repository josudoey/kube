## kube-pod-watcher usage

```
$ go install github.com/josudoey/kube/cmd/kube-pod-watcher@latest
$ kube-pod-watcher -h
$ kube-pod-watcher
```


## kube-vhost usage

```
$ go install github.com/josudoey/kube/cmd/kube-vhost@latest
$ kube-vhost -h
$ kube-vhost show
$ kube-vhost server --port 8010
```
